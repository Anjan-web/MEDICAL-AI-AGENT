# Medical AI Agent

A production-ready medical question-answering system built with RAG, LangGraph, FastAPI.

## Features
- Encyclopedia RAG (26,512 chunks, PubMedBERT embeddings)
- Live PubMed research integration
- WHO/CDC guidelines via PubMed filter
- FastAPI REST endpoint
